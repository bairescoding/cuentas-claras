# 🚀 Blazor WASM PWA Vibe Coding Template

Bienvenido a la plantilla de Arquitectura Élite para proyectos **Blazor WebAssembly PWA (Single Page Applications)**. Este repositorio está completamente optimizado para entornos de Vibe Coding multi-agente, integrando reglas estrictas para AI, CI/CD automatizado, y directrices de arquitectura.

## 🤖 Soporte Multi-Agente
Este template incluye directrices y contexto pre-configurado (Skills) para mantener alineados a distintos agentes autónomos:
- **Cursor / Claude Code**: `.cursor/rules/` y `claudecode.toml`
- **Google Antigravity**: `.agent/`
- **OpenCode**: `.opencode/`
- **GitHub Copilot**: `.github/copilot-instructions.md`
